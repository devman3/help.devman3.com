Add an attachment to a bursary profile

On the Bursary summary page, select Edit.

Scroll down until you see Attachments on the left and click on Add on the left.

Select the date the document was submitted. 

Select the type of document from the drop-down list.

Click on Choose file to upload the document then click on Save.

Add a description or name for the document.

You can now see the newly attached document under Attachments. 

You can click on Add to upload more documents or Save and close.
