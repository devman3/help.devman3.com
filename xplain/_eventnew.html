Add a new event

Select the Events tab, then click on Add.

Select the date the event will be held. 

Type in the name of the event. 

Select a campaign if the event is linked to one. 

Select a subscription if applicable. 

Click here to enter the maximum number of guests. 

Select the type of event. 

Select the manager of the event. 

Click through to select the RSVP expiry date. 

Select the web stationery, then click on Save and stay on page.

Event notes, documentation etc. can be uploaded here. 

As you invite guests, the number and status of your invitees will populate. You can click here to add invitees or manage their status. 

Create a new questionnaire template by clicking in Add. You can also view and personalise your online questionnaire. 

Select Save and close.
